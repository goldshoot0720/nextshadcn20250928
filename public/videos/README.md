## 注意事項：

- 支援的影片格式：MP4, WebM, OGG, AVI, MOV
- 建議使用 MP4 格式以獲得最佳兼容性
- 影片會自動快取到用戶的本地存儲以減少流量使用
- 系統支援斷點續傳和流式播放

## 快取機制：

系統使用 IndexedDB 來存儲影片文件到用戶的瀏覽器本地存儲中：
- 首次播放時會提示下載到本地
- 後續播放直接從本地快取讀取
- 大幅減少網路流量使用
- 提升播放體驗和載入速度
