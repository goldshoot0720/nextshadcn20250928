# 歌詞排版優化說明

## 改進內容

### 1. 智能歌詞格式化
- **結構化歌詞**: 自動識別 `[Verse]`, `[Chorus]`, `[Bridge]` 等標籤，並美化顯示
- **敘事歌詞**: 智能分段長篇連續文本，提高可讀性
- **普通歌詞**: 優化行間距和字體顯示

### 2. 視覺美化
- **漸變背景**: 使用漸變色背景提升視覺效果
- **關鍵詞高亮**: 自動高亮 "塗哥"、"鋒兄"、"水電工" 等關鍵詞
- **段落標籤**: 美化的段落分隔符和標籤顯示
- **響應式設計**: 適配不同屏幕尺寸

### 3. 交互功能
- **字體大小調節**: A+/A- 按鈕調整歌詞字體大小 (12-24px)
- **歌詞內搜索**: 在當前歌詞中搜索特定文字，高亮顯示
- **複製功能**: 一鍵複製歌詞到剪貼板
- **導出功能**: 支持導出為 TXT 和 Markdown 格式

### 4. 多語言支持
- **中文**: 繁體中文歌詞
- **English**: 英文翻譯版本
- **日本語**: 日文翻譯版本  
- **粵語**: 粵語版本

## 歌詞格式示例

### 結構化格式
```
[Intro]
歌曲開場白

[Verse 1]
第一段歌詞內容

[Chorus]
副歌部分

[Bridge]
橋段

[Outro]
結尾
```

### 敘事格式
長篇連續文本會被智能分段，根據句號、感嘆號等標點符號和關鍵詞自動分段。

## 技術特點

### CSS 樣式
- 使用 CSS-in-JS 實現動態樣式
- 支持深色模式
- 流暢的懸停效果和過渡動畫

### React 組件
- 使用 `useCallback` 優化性能
- 智能的歌詞格式檢測
- 響應式的字體大小控制

### 用戶體驗
- 直觀的操作界面
- 實時搜索高亮
- 無縫的語言切換

## 使用方法

1. **選擇歌曲**: 從左側列表選擇要查看的歌曲
2. **切換語言**: 使用頂部標籤切換不同語言版本
3. **調整字體**: 使用 A+/A- 按鈕調整字體大小
4. **搜索歌詞**: 在搜索框中輸入要查找的文字
5. **複製導出**: 使用複製或導出按鈕保存歌詞
