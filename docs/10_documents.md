# 鋒兄文件 (Documents)

## 功能概述
綜合文件管理系統，專為各種常用辦公文件與壓縮檔提供上傳、預覽與下載功能。

## 主要特點
- **廣泛檔案支援**：支援 PDF、TXT、Word (`.docx`)、Excel (`.xlsx`)、PowerPoint (`.pptx`)、ZIP 檔案。
- **媒體預覽**：新增圖片 (`.jpg`, `.png`, `.gif`, `.webp`)、音訊 (`.mp3`, `.m4a`, `.wav`)、影片 (`.mp4`, `.webm`) 的即時預覽功能。
- **Office 檔案在線預覽**：透過 Microsoft Office Web Viewer 可直接瀏覽 Word、Excel、PowerPoint 檔案。
- **ZIP 結構預覽**：點擊預覽 ZIP 檔案，可瀏覽內部檔案結構與大小。
- **直接下載**：點擊下載按鈕強制觸發檔案下載，避免瀏覽器預覽二進制內容。
- **分類管理**：支援自訂分類，快速過濾與組織文件。
